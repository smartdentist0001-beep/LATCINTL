# LATCINTL - Pi Network Marketplace

Premium African Fabrics & Fashion Store built for **Pi Browser & App Studio**.

## Features
- Full Pi Authentication
- Pi Payments (U2A) - Testnet + Mainnet ready
- Beautiful responsive UI (React + Tailwind)
- Shopping Cart
- Category & Search filters
- Production-ready backend

## Tech Stack
- **Frontend**: React + Vite + Tailwind CSS
- **Backend**: Vercel Serverless Functions (Node.js)
- **Payments**: Official Pi SDK
- **Deployment**: Vercel

npm install
