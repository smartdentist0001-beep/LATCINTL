## Overview

LATCINTL is a Next.js marketplace dApp for selling fabrics, atamfa, and clothing on Pi Network. It supports:

- ✅ Pi Authentication (buyer identification)
- ✅ Pi Payments (Testnet & Mainnet)
- ✅ Webhook callbacks (payment status updates)
- ✅ Order tracking
- ✅ Admin product management
- ✅ Pi Browser & Pi App Studio compatible
